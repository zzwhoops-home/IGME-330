I. 2 Audio Effect nodes

I added a StereoPannerNode and a WaveShaperNode, which I read online can do panning left and right and apply a distortion effect, respectively. Both are additional sliders, with the panner going between [-1, 1] and the WaveShaper having a custom makeDistortionCurve(amount) function which I found on mdn. All I know is that this creates an array of floats which tell the WaveShaperNode how to alter the waveform, which in this case produces a distortion effect.

II. Sprite

My TeslaSprite class is located within src/sprite.js, and I made an animation that should be reminiscent of a Tesla coil. While it does not arc to anything on screen, there are several things which are affected by the average volume. The average volume is determined by two parameters, dataStart and dataEnd. The audioData is sliced between these values, 0 being the beginning of the data and 1 being the end, and its values are averaged. These attributes of the "Tesla coil" are affected:

  • Radius of arcs - how far the arcs travel
  • Radius variance - how much the arc travel distance varies
  • Number of arcs
  • y-distance - this will act as if it is "traveling" with the rectangles behind
  • Alpha value of lines - a smaller value will make the lines fade out more

By default, I have placed two coils, one on each side. The left coil corresponds to the lower frequencies between 0 and 0.5, and the right coil corresponds to the higher frequencies between 0.5 and 1.0. However, if you use the "Coil Slider", you can change the data range that is averaged.
